## Notes & Decisions

- **"Cake Slice" on the ticket = Frosted Pieces (`fr_*`) in the system** — confirmed by user 2026-04-19
- **"Bread Pudding Slice" = `pz_pudin`** (Pieces category, not Plain) — confirmed by user 2026-04-19
- The `_nt` suffix on product keys means "no topping" variant
- Round cakes have position variants: `inside`, `top`, `inside_nt`, `top_nt`
- Standard/frosted/pieces items have base key + `_nt` variant
- **Always confirm new ticket code mappings with the user before using them in production**
