## Overview
This file tracks future capabilities for the Cecilia Bakery admin ecosystem, ensuring good ideas aren't forgotten and providing a roadmap for future expansion once core features are stabilized.

## Feature Wishlist / Roadmap

### 1. Dashboard Analytics Hub (Reporting Suite) — 🔄 IN PROGRESS
See [`IMPLEMENTATION_INSIGHTS.md`](./IMPLEMENTATION_INSIGHTS.md) for the full plan.
- [x] **Advanced Analytics:** A dedicated Insights tab with donut charts, revenue breakdowns, and driver leaderboard.
- [ ] **Action Items View:** A dedicated, full-screen page for pending tasks (missing driver EODs, Wholesale approvals, unattended online orders).
- [ ] **Custom Date Ranges:** Ability to filter the overview and reports by custom date ranges, not just "Today" or "This Month".

### 2. Wholesale Portal
- [ ] Self-serve portal for B2B wholesale clients to place and track bulk orders.
- [ ] Custom pricing tiers applied per wholesale account.

### 3. Production Tracking (Kitchen Display)
- [ ] A dedicated UI view for the kitchen staff showing real-time production requirements based on incoming driver and online orders.
